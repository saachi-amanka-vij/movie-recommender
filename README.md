# movie-recommender
Advanced movie recommendation system using TF-IDF, cosine similarity, clustering, and context-aware personalization
Movie Recommendation System | Python, Scikit-learn, Streamlit
• Built a content-based movie recommender using TF-IDF and cosine similarity
• Implemented explainable AI by extracting top keywords for each recommendation
• Added context-aware filtering using mood and weather inputs
• Applied K-Means clustering for unsupervised movie grouping
• Developed an interactive UI using Streamlit for real-time recommendations
